# Accessibility testing bookmarklet that works with the Shadow DOM
[Get the bookmarklet](https://jasonday.github.io/VA11y/)

All of the bookmarklets have been reworked into a single testing tool - VA11y
- Accessible name calculation (Estimated. May not work with all permutations of Shadow DOM implementation, custom components, etc.). Focus on elements to see accessible name calculation, as well as role and state.
- Image reporting. Hover over images to view image text alternatives, if decorative, or missing alt text
- Headings. View heading structure and missed heading levels, empty headings, etc.
- Tools
    - Page structure. View HTML5 landmarks and other structural elements.
    - Text spacing. Apply text spacing to test WCAG 1.4.12
