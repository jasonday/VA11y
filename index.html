Accessibility testing bookmarklets

Bookmarklets for improved accessibility testing with Shadow DOM/Web components support.

Bookmarklets

To use a bookmarklet, save it as a bookmark by dragging the link to your bookmarks bar or by manually creating a new bookmark and pasting the JavaScript code into the URL field. Once saved, navigate to the webpage you want to use it on and click the bookmarklet from your bookmarks bar to execute the code.

aName

aName reports an approximation of the accessible name of the focused element. It returns the text label, where that's defined, the focused element type, any attached descriptions, and the current state if applicable. For shadow DOM components, it looks at the focused element and the host element to determine attribute state and will attempt to get the accessible name from slot references. Note: this may not be consistent or work for all permutations of shadow DOM components.
